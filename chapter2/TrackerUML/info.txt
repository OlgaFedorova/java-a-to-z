Вся программа располагается в пакете ofedorova.
Пакет ofedorova включает в себя:
1. Пакет commands - содержит классы команд консольного меню. Я попыталась реализовать паттерн "Command".
Класс Command - абстрактный класс, реализующий абстрактную команду.
Класс AddNewTaskCommand наследник класса Command, реализует команду "Добавление новой заявки".
Класс EditTaskCommand наследник класса Command, реализует команду "Редактирование заявки".
Класс RemoveTaskCommand наследник класса Command, реализует команду "Удаление заявки".
Класс ShowListTasksCommand наследник класса Command, реализует команду "Отображение списка всех заявок".
Класс ShowListTasksFilterCommand наследник класса Command, реализует команду "Отображение заявок с условием фильтра".
Класс AddNewCommentCommand наследник класса Command, реализует команду "Добавлять к заявки комментарий".
Класс ExitCommand наследник класса Command, реализует команду "Выход из программы".

2.Класс Task описывает модель заявки.
Класс содержит поля name (Имя), description (Описание), dateCreate (Дата создания), listComments (список комментариев), геттеры и сеттеры для этих полей.

3. Класс ConsoleHelper - класс содержит статические методы для работы с консолью.
Метод readCommand - считывает с консоли текст и возвращает соответсвующую команду.
Метод readString  - считывает с консоли и возвращает строку.
Метод writeString - выводит переданную в метод строку в консоль.

4. Класс Tracker  - главный класс программы, который содержит метод main и основную логику работы.
Поле listTasks - список заявок.
Поле exit - признак того, что пора выйти из программы.
Метод getListTasks - геттер для поля listTasks, геттер будет возвращать unmodifiableList.
Метод addTask - добавляет заявку в список.
Метод isExit - геттер для поля exit.
Метод setExit - сеттер для поля exit.
Метод introduce - выводит в консоль приветсвисе и информацию о программе.
Метод showConsoleMenu - выводит консольное меню.

5. Код метода main класса Tracker будет следующий:

public static void main (String[] args){
	
	Tracker tracker = new Tracker(); //создаем объект главного класса
	tracker.introduce(); // Выводим на печать информацию о программе
	
	do{
	
		tracker.showConsoleMenu(); //показываем косольное меню
		Command command = ConsoleHelper.readCommand(); //получаем команду от пользователя
		command.excute(); // Выполняем команду
	
	}
	while(!tracker.isExit()); //пока не сказали "Выходи" выполняем тело цикла

}
